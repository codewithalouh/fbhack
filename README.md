## Installing

To install, follow these steps:

1. Clone the repository: 
```
git clone https://github.com/codewithalouh/fbhack
```
2. Navigate to the root directory of the project: 
```
cd fbhack
```

## Usage

To run, execute the setup script: 
```
python3 setup.py
```
